# vueCli3

## 首次运行说明

```
$ git clone xxx
$ npm install
<!-- 注:如果直接切换分支并拉取代码 需删除原有node_modules -->
<!-- 注:统一使用node 8.9.3 因为node 8.9.3与node 10.14.2  环境下安装的node-sass会有兼容性问题-->
<!-- 注:推荐使用nvm node版本管理器-->
```

## 启动
```
$ npm install
$ npm run serve
```

## 打包测试环境
```
$ npm run test
```

## 打包正式环境
```
$ npm run build
```
